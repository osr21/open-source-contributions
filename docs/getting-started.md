# Getting Started

## Prerequisites
- Node.js 18+
- pnpm or npm
- A GitHub account

## Installation
```bash
git clone https://github.com/osr21/open-source-contributions
cd open-source-contributions
npm install
```

## Running Locally
```bash
npm run dev
```

## First Contribution
Check the issues labeled `good first issue` to find a good starting point.
