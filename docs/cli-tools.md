# Useful CLI Tools for Developers

## File & Directory

| Tool | Install | Purpose |
|------|---------|---------|
| `exa` | `npm i -g exa` | Modern `ls` replacement with icons |
| `bat` | `brew install bat` | `cat` with syntax highlighting |
| `fd` | `brew install fd` | Fast `find` alternative |
| `ripgrep` (`rg`) | `brew install ripgrep` | Ultra-fast grep |
| `fzf` | `brew install fzf` | Fuzzy file finder |
| `tree` | `brew install tree` | Directory tree view |
| `dust` | `brew install dust` | Disk usage viewer |

## Network

| Tool | Install | Purpose |
|------|---------|---------|
| `curl` | built-in | HTTP requests |
| `httpie` | `brew install httpie` | Human-friendly HTTP client |
| `wget` | `brew install wget` | File downloads |
| `nmap` | `brew install nmap` | Network scanner |
| `netcat` (`nc`) | built-in | TCP/UDP debugging |

## Development

| Tool | Install | Purpose |
|------|---------|---------|
| `jq` | `brew install jq` | JSON processor |
| `yq` | `brew install yq` | YAML processor |
| `gh` | `brew install gh` | GitHub CLI |
| `tldr` | `npm i -g tldr` | Simplified man pages |
| `nvm` | script | Node version manager |
| `fnm` | `brew install fnm` | Fast Node version manager |

## Git

| Command | Purpose |
|---------|---------|
| `git log --oneline --graph` | Visual branch history |
| `git stash` | Temporarily shelve changes |
| `git bisect` | Binary search for a bug |
| `git reflog` | Recover lost commits |
| `lazygit` | Terminal UI for git |
